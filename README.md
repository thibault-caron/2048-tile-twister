# 2048-tile-twister
Simple Tile Twister game in C++

In case you encounter error "build.sh" appear and you're on windows, launch this command to go from CRLF to LF (command on git bash!)
```bash
dos2unix build.sh
```

Create executable :

```bash
docker compose up windows --build
```

Execute executable
```bash
./build/2048.exe
```
